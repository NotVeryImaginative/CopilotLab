# Setting Up GitHub Copilot in PyCharm on macOS

## Prerequisites
- PyCharm (Professional or Community Edition) installed
- GitHub account with Copilot subscription

## Steps

1. **Open PyCharm**  
   Launch PyCharm on your Mac.

2. **Install Copilot Plugin**
   - Go to `Preferences` (`Cmd + ,`)
   - Navigate to `Plugins`
   - Search for "GitHub Copilot"
   - Click `Install` on the GitHub Copilot plugin
   - Restart PyCharm if prompted

3. **Sign In to GitHub**
   - After restart, you may see a prompt to sign in to GitHub.
   - Click `Sign In` and follow the authentication steps in your browser.

4. **Enable Copilot**
   - Go to `Preferences` > `Tools` > `GitHub Copilot`
   - Ensure Copilot is enabled

## Troubleshooting

- If you do not see suggestions, check your internet connection and ensure your Copilot subscription is active.
- Update PyCharm and the Copilot plugin to the latest versions if issues persist.
